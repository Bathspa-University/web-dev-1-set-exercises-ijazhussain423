Welcome to the HTML & CSS Quick-Reference Guide

This guide covers essential HTML and CSS tags and properties.

HTML Tags

  • <div>: Defines a division or a section in an HTML document. 
    <div>
    Content goes here
</div>
  • <p>: Defines a paragraph. 
    <p>This is a paragraph.</p>
  • <a>: Defines a hyperlink. 
    <a href="https://example.com">Visit our website</a>
  • <img>: Defines an image. 
    <img src="image.jpg" alt="Description">
  • <ul>: Defines an unordered list. 
    <ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

CSS Properties

  • color: Specifies the text color. 
    color: blue;
  • font-family: Specifies the font family. 
    font-family: Arial, sans-serif;
  • background-color: Specifies the background color. 
    background-color: #f0f0f0;
  • padding: Specifies the padding inside an element. 
    padding: 10px;

That's it for our quick-reference guide! Happy coding!
